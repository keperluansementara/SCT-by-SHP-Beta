# 🚀 SCT by SHP v6

**Sales Control Tower (SCT)** adalah dashboard Business Intelligence untuk industri FMCG yang dikembangkan untuk membantu monitoring performa penjualan, Customer Active, Wholesaler, Modern Trade, dan Executive Decision dalam satu dashboard interaktif.

> Internal Project

Sales Control Tower (SCT) developed by Suryo Hadi Prakoso.

This repository is intended for internal development and maintenance.
The source code and business logic are proprietary.

---

## ✨ Features

### Executive Dashboard
- Executive Summary
- Cross Section Analysis
- Executive Decision
- Priority Action

### Sales Performance
- Performance Overview
- Performance by Region
- Performance by Principle
- Performance by Channel

### Customer Active
- CA Achievement
- CA Recovery Opportunity
- CA Trend Analysis

### Wholesaler
- Performance by Program
- Performance by Class
- Timegone Analysis
- Opportunity Analysis

### Modern Trade
- Executive Summary
- Channel Analysis
- Timegone Analysis
- Decision Analysis

### PS Achiever
- Sell In
- Sell Out
- Achievement Analysis

### Export
- Export PDF
- Export PNG
- Copy to Clipboard

### Integration
- Excel Upload
- Google Drive Refresh
- Runtime Health Check
